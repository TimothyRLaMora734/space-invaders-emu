# SpaceInvaders-Emu

An Intel 8080 Emulator capable of playing a Space Invaders ROM written in C++

## Dependencies
* [SDL 2](https://www.libsdl.org/index.php)
* [libzip](https://nih.at/libzip/)
* C++14 compiler
